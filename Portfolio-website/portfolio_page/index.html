Progress So Far

At the beginning, I was really confused about how to use flexbox, especially when it came to centering elements. I used to mix up properties like flex-direction and justify-content, but now I understand better how they work together for layout and alignment.

Read more

I’ve also recently started learning CSS grid. At first, I thought it was simple, but now I see that it’s different from flexbox and requires a different way of thinking about layout. I can create basic structures with it, but I’m still learning how to properly apply it to real page sections like headers, main content, and footers.

Another area I’ve improved in is understanding the relationship between parent and child elements in CSS. I now know how styling a parent can affect its children, and how to target elements more intentionally.

I still get confused when moving from writing HTML to styling with CSS especially when deciding how I want the design to look. Sometimes I also misuse spacing properties like margin and padding, and only realize later that I could have used a better approach.

For practice, I consistently build small projects and try to recreate them across different environments. For example, I use my laptop and also practice on my phone using CodePen. I’m focused on improving through repetition and hands-on practice, and I’m committed to getting better over time.

Blockers

One of my biggest challenges is applying what I have learned to real projects. For example, when I learned float layouts, I understood it in practice, but when I tried to use it in my own webpage, it did not behave the way I expected.

Read more

I had issues with placing images and getting text to wrap properly around them. Sometimes the text would overlap or not align correctly, and I ended up adjusting things manually without fully understanding why it worked. That makes it hard for me to confidently recreate the same result again.

I also struggle with moving from writing HTML to actually styling the page. After setting up the structure, I often feel confused about how to approach the layout and design like where elements should go and how to arrange them properly.

Another challenge is understanding when to use certain CSS properties. For example, I sometimes misuse margin and padding, or I am unsure which element to style like whether to style a button itself or the link inside it.

I also notice that when I watch tutorials, I understand while watching, but when I try to practice later, I struggle to remember the exact steps or apply the concepts on my own.

Overall, I think my main blocker is that I need a deeper understanding of CSS concepts so I can apply them confidently, instead of guessing or relying on trial and error.
